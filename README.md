# Student Record Management System (Python)

A simple console-based CRUD (Create, Read, Update, Delete) application for managing student records.  
This project was built to demonstrate fundamental programming logic, file handling, and user interaction in Python.

---

##  Features
- Add new student records (Name, ID, and Grade)
- View all student records
- Update existing records
- Delete student records
- Exit the program safely

---

##  Technologies Used
- Python  
- Command-line interface (CLI)
